# Blog Frontend

Frontend para el blog del desarrollador Valentí Gàmez.

## Sobre mi

https://valentigamez.com

## Instalación

``` bash
# install dependencies
$ npm run install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
